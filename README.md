# Task Management System - RESTful APIs using Node.js

## Technologies:

- Node.js (Express)
    1. Install Dependencies
    
        ```
        npm install
        ```
    2. Run App
    
        ```
        npm start
        ```
- PostgreSQL
    1. Docker Build
    
        ```
        docker build -t pg-db . 
        ```
    2. Docker Run
    
        ```
        docker run -d -p 5432:5432 pg-db
        ```
